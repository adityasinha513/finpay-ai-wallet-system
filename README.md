# FinPay – AI-Powered Digital Wallet System

Production-grade fintech backend system built using Java and Spring Boot.

## Features
- JWT Authentication
- Wallet Management
- Peer-to-Peer Transfers
- Redis Caching
- Kafka Event Streaming
- AI Financial Assistant
- Fraud Detection Agent
- Dockerized Deployment
- Monitoring & Observability

## Tech Stack
- Java 21
- Spring Boot
- PostgreSQL
- Redis
- Kafka
- Docker
- Spring AI
